Overview

render_farm_filter(String $filter) ⇒ String

Render a farm filter entry. Will check if it needs to render regex with single quote or regular strings with double quotes.

Examples:

Rendering globs

render_farm_filter("*")

Rendering regex

render_farm_filter("'(content|apps).*'")

Parameters:

  • filter (String) —

    Filter string to render

Returns:

  • (String) —

    Regex or regular string farm entry

Render a farm filter entry.
Will check if it needs to render regex with single quote or regular strings with double quotes.
@param filter [String] Filter string to render
@return [String] Regex or regular string farm entry
@example Rendering globs
  render_farm_filter("*")
@example Rendering regex
  render_farm_filter("'(content|apps).*'")

  ) do |args|
  # Validate input
  unless args.length == 1
